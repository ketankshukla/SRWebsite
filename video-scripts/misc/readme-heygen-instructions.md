# HeyGen Video Creation Guide for Surplus Funds Course

## üé¨ Step-by-Step Process for Each Chapter

### Step 1: Log into HeyGen

1. Go to [app.heygen.com](https://app.heygen.com)
2. Click **"Create Video"**

### Step 2: Choose Your Method

#### Option A: Avatar Video (Recommended)

1. Click **"Avatar Video"**
2. Select **"Create from Script"**
3. Choose your avatar (recommended: professional business look)
4. Choose voice that matches avatar

#### Option B: Template-Based

1. Click **"Templates"**
2. Search for "Education" or "Course" templates
3. Select one and customize

### Step 3: Add Your Script

1. Open the chapter script file (e.g., `chapter-01-script.txt`)
2. Copy the content between scene markers
3. Paste into HeyGen's script box
4. Each `[SCENE]` marker = new slide/scene in HeyGen

### Step 4: Customize Each Scene

For each scene:

1. **Background**: Choose office, gradient, or upload custom
2. **Text overlays**: Add key points as on-screen text
3. **Images/Graphics**: Upload relevant visuals
4. **Transitions**: Add smooth transitions between scenes

### Step 5: Generate & Download

1. Click **"Submit"** or **"Generate"**
2. Wait for processing (usually 5-15 minutes)
3. Download in 1080p
