How to Create Your OG Image

  1. Open this file in Chrome browser
  2. Right-click on the image above and select "Inspect"
  3. In DevTools, right-click on the <div class="og-image"> element
  4. Select "Capture node screenshot"
  5. Save the file as og-image.png in your images folder

Tip: The image is 800x800 pixels - a square format.
